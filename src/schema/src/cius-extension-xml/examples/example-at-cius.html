Introduction

This CIUS-AT-NAT builds on top of the European Norm EN 16931-1:2017. All rules of the underlying specification apply with the addition of the rules stated in this document.

Indetifying philip.helger@brz.gv.at

An identification of the specification containing the total set of rules regarding semantic content, cardinalities and business rules to which the data contained in the instance document conforms is reported in BT-24 “Specification identification”. The identifier for this CIUS is:

BT-24 1..1 Specification identification urn:cen.eu:en16931:2017#compliant#urn:erechnung.gv.at:CIUS-AT-NAT:1.0.0

According to the e-Invoice instance document syntax the specification identification will be:

UBL 2.1:

<cbc:CustomizationID>urn:cen.eu:en16931:2017#compliant#urn:erechnung.gv.at:CIUS-AT-NAT:1.0.0</cbc:CustomizationID>

CII 16B:

<rsm:ExchangedDocumentContext> 
  <ram:GuidelineSpecifiedDocumentContextParameter>
    <ram:ID>urn:cen.eu:en16931:2017#compliant#urn:erechnung.gv.at:CIUS-AT-NAT:1.0.0</ram:ID>
  </ram:GuidelineSpecifiedDocumentContextParameter>
</rsm:ExchangedDocumentContext>

New rules imposed by this specification

[BR-AT-1] Postal address line 1 becomes mandatory [BT-35, BT-50, BT-64, BT-75]
[BR-AT-2] Postal address city becomes mandatory [BT-37, BT-52, BT-66, BT-77]
[BR-AT-3] Postal address post code becomes mandatory [BT-38, BT-53, BT-67, BT-78]
[BR-AT-4] Actual delivery date or invoicing period must be provided [BT-72, BT-73, BT-74]
[BR-AT-5] Document level allowance VAT rate becomes mandatory [BT-96]
[BR-AT-6] Document level charge VAT rate becomes mandatory [BT-103]
[BR-AT-7] External document location is not supported [BT-124]
[BR-AT-8] The size of all attached documents together may not exceed 15MB []
[BR-AT-9] Invoiced item VAT rate becomes mandatory [BT-152]
